# Норма матрицы (1-норма)

Данная программа считывает матрицу размером `N × M` и вычисляет **Столбцовая норма (1-норма)** — максимальную сумму модулей элементов по столбцам.

$$
\|A\|_1 = \max_j \sum_i |a_{ij}|
$$

## Алгоритм

1. Считать размеры матрицы `N` и `M`.
2. Считать элементы матрицы $A[i][j]$.
3. Для каждого столбца `j`:
   - вычислить сумму $S_j = \sum_{i=1}^{N} \lvert A[i][j] \rvert$
4. Найти максимальную из сумм $S_j$.
5. Вывести найденное значение как $\|A\|_1$.

## Использование

1. Введите длину матрицы (N)
2. Введите ширину матрицы (M).
3. Введите элементы матрицы.ы

## Пример

Для матрицы

$$
\begin{bmatrix}
1 & -2 \\
3 & 4
\end{bmatrix}
$$

суммы по столбцам:

- $|1| + |3| = 4$
- $|-2| + |4| = 6$

Следовательно, $\|A\|_1 = 6$.

## ⚠️ Ограничения текущей реализации

- ❌ Массив фиксированного размера `20×20`  
- ❌ Начальное значение `max = 0` даёт неверный результат при отрицательных числах
