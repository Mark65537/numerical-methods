# Метод вращения (Givens) для СЛАУ

Небольшой учебный проект на C++, который решает систему линейных алгебраических уравнений методом вращений.

## Что делает программа

- считывает расширенную матрицу из файла `mtx.txt`;
- выполняет преобразование матрицы методом вращений;
- выводит полученные корни (`roots`);
- считает и печатает вектор невязки.

В коде используется фиксированная размерность:

- `n = 5` (число уравнений),
- `m = n + 1` (расширенная матрица).

## Формат `mtx.txt`

Файл должен содержать `n` строк по `m` чисел в каждой строке.

Пример для текущей версии программы:

```txt
a11 a12 a13 a14 a15 b1
a21 a22 a23 a24 a25 b2
a31 a32 a33 a34 a35 b3
a41 a42 a43 a44 a45 b4
a51 a52 a53 a54 a55 b5
```

## Примечания

- Проект использует `windows.h` и кодировку консоли CP1251.
- Если меняете размер системы, обновите константу `n` в `Main.cpp` и входной файл `mtx.txt`.
