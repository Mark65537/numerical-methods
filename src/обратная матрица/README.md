Программа для вычисления обратной матрицы методом алгебраических дополнений.

## Описание

Программа генерирует случайную квадратную матрицу заданного размера и вычисляет её обратную матрицу.

### Возможности

- Генерация случайной матрицы n×n
- Вычисление определителя матрицы
- Вычисление обратной матрицы (если определитель ≠0)
- Вывод исходной и обратной матриц

### Алгоритм

Обратная матрица вычисляется по формуле:

```
A⁻¹ = (1/det(A)) * adj(A)
```

где:

- `det(A)` — определитель матрицы
- `adj(A)` — присоединённая (союзная) матрица

### Cборка

Откройте `обратная матрица.sln` в Visual Studio и соберите проект (Ctrl+Shift+B).

## Использование

1. Запустите скомпилированный исполняемый файл
2. Введите размер матрицы n
3. Программа выведет:
    - Сгенерированную матрицу
    - Определитель
    - Обратную матрицу (если она существует)

Если определитель равен 0, обратная матрица не существует.
