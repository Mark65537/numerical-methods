# матрица Гилберта и вращение

Консольная программа на C++, реализующая численные методы для работы с **матрицей Гилберта**:

- построение матрицы
- вычисление обратной матрицы
- оценка нормы
- вычисление числа обусловленности

## Что делает программа

Программа строит **матрицу Гилберта** размера `k × k`:

$$
H_{ij} = \frac{1}{i + j + 1}, \quad i,j = 0..k-1
$$

Затем вычисляет:

- **обратную матрицу** $H^{-1}$;
- **норму** $\|H\|$ и $\|H^{-1}\|$ (используется оценка по строкам: берётся максимум по строкам от суммы элементов строки);
- **число обусловленности** $q = \|H\|\cdot\|H^{-1}\|$.

## Используемые методы

Решение СЛАУ выполняется с помощью метода вращений (Givens rotations), который:

- приводит матрицу к верхнетреугольному виду
- устойчив к численным ошибкам

## Норма матрицы

Используется оценка нормы по строкам:

$$
||A|| = \max_i \left| \sum_{j} A[i][j] \right|
$$

## Функциональность

- Генерация матрицы Гилберта
- Вывод матрицы в консоль
- Вычисление нормы матрицы
- Построение обратной матрицы
- Проверка (A × A⁻¹ ≈ I)
- Вычисление числа обусловленности

## Структура проекта

- `Main.cpp`: генерация матрицы Гилберта, печать, вычисление нормы, сборка обратной матрицы по столбцам, расчёт \(q\).
- `Вращение.h`: реализация вращений Гивенса (`alg`) + обратная подстановка (`alg_2`).

## Важно

- Матрица Гилберта **плохо обусловлена**, особенно при увеличении размера `k`
- При больших `k` возможны значительные ошибки округления
- Используется динамическое выделение памяти (`new`), освобождение памяти не реализовано
