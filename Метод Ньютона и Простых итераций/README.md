# Метод Ньютона (касательных) и Метод простой итерации

В программе реализованы два численных метода решения нелинейного уравнения:

$$
f(x) = \sin(x^2) - 6x + 1 = 0
$$

## Описание и алгоритм

### 1. Метод Ньютона (касательных)

Метод основан на использовании касательной к графику функции в текущей точке приближения.

Итерационная формула:
$$
x_{k+1} = x_k - \frac{f(x_k)}{f'(x_k)}
$$

В данной реализации производная вычисляется численно методом конечных разностей:
$$
f'(x) \approx \frac{f(x + h) - f(x)}{h}
$$

Алгоритм:

1. Задать начальное приближение $x_0$
2. Вычислить производную
3. Найти новое значение $x_{k+1}$
4. Повторять, пока:
   $$
   |x_{k+1} - x_k| > \varepsilon
   $$

Особенности:

- Быстрая сходимость (квадратичная)
- Требует вычисления производной
- Чувствителен к начальному приближению

---

### 2. Метод простой итерации

Исходное уравнение приводится к виду:
$$
x = \varphi(x)
$$

В программе используется преобразование:
$$
\varphi(x) = \frac{\sin(x^2) + 1}{6}
$$

Итерационная формула:
$$
x_{k+1} = \varphi(x_k)
$$

Алгоритм:

1. Задать начальное приближение $x_0$
2. Вычислить $x_{k+1} = \varphi(x_k)$
3. Повторять, пока:
   $$
   |x_{k+1} - x_k| > \varepsilon
   $$

Особенности:

- Простота реализации
- Сходимость зависит от выбора $\varphi(x)$
- Обычно медленнее метода Ньютона

---

## Использование

1. Задайте параметры в коде:
   - начальное приближение `x0`
   - точность `eps`

    ```cpp
    x0 = 0.1;
    eps = 0.000001;
    ```

2. Скомпилируйте и запустите программу.
3. Программа выведет:
    результат метода Ньютона
    результат метода простой итерации

## Примечания

- Для корректной работы метода Ньютона рекомендуется пересчитывать производную на каждой итерации
- Условие остановки должно быть:
    ```cpp
    while (abs(x - x0) > eps);
    ```
- Метод простой итерации сходится только при выполнении условия сжатия:
  $∣φ′(x)∣<1$
