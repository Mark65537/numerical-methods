# Interpolation 2

Программа выполняет численную интерполяцию двух заданных функций:

- полиномом Лагранжа
- кубическим сплайном (кусочно-кубическая функция)

Исходный код находится в папке `Interpolation 2/Interpolation 2/` (входная точка: `Source.cpp`).
Входные данные задаются прямо в коде (интервалы, формулы функций, число узлов `N` и шаг сетки `U=1000`).
Выходные данные программы — таблицы значений в файлах `*.txt`, которые можно открыть в Excel.

## Алгоритм

1. Для задания `q=1` и `q=2` выбираются:
   1) функция `f(x)`
   2) отрезок интерполяции `[a, b]`
   3) число узлов интерполяции `N` (равномерная сетка)
   4) сетка для оценки интерполяции: `U=1000` точек между `a` и `b`.
2. Узлы интерполяции строятся так:
   - `x_i = a + i*(b-a)/(N-1)`
   - `y_i = f(x_i)`
3. Полином Лагранжа:
   - для каждой точки `t` вычисляется
     `P(t) = sum_{i=0..N-1} y_i * prod_{j!=i} (t - x_j)/(x_i - x_j)`
4. Кубический сплайн:
   - коэффициенты сплайна строятся по узлам `x_i, y_i`
   - значения коэффициентов находятся решением СЛАУ для трёхдиагональной матрицы методом прогонки (прямой/обратный ход)
   - затем сплайн вычисляется на нужном отрезке (сегмент выбирается бинарным поиском) по формуле кусочного кубического многочлена
5. Для каждой точки оценочной сетки программа записывает `x`, точное значение `f(x)` и значение интерполяции в `*.txt`.

## Использование

1. Программа автоматически выполнит оба задания (`q=1` и `q=2`) и выведет в консоль несколько промежуточных значений.
2. После завершения появятся/обновятся файлы `*.txt` в рабочей директории (их можно построить в Excel по колонкам `x`, `f(x)`, `interp(x)`).

## Выходные данные

Программа создаёт (перезаписывает) табличные файлы `*.txt`.

Формат каждой строки: табуляция-разделённые колонки:
- `x` — точка на сетке
- `f(x)` — значение исходной функции
- `interp(x)` — значение интерполяции в точке `x`

Список файлов, которые генерируются из кода:

- `Lagrang_5.txt` — полином Лагранжа, `f1(x)`, `N=5`, `[a,b]=[-1,1]`
- `Lagrang_20.txt` — полином Лагранжа, `f1(x)`, `N=20`, `[a,b]=[-1,1]`
- `Lagrang2_20.txt` — полином Лагранжа, `f2(x)`, `N=20`, `[a,b]=[0,PI]`
- `Spline_5.txt` — кубический сплайн, `f1(x)`, `N=5`, `[a,b]=[-1,1]`
- `Spline_20.txt` — кубический сплайн, `f1(x)`, `N=20`, `[a,b]=[-1,1]`
- `Spline2_20.txt` — кубический сплайн, `f2(x)`, `N=20`, `[a,b]=[0,PI]`

Используемые функции в коде:

- `f1(x) = 1 / (1 + 25*x^2)`
- `f2(x) = exp(sin(x)^2) * sin(3*x)`
