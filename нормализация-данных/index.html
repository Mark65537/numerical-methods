Учебный проект по численным методам: генерация матрицы случайных чисел и ее нормализация по максимальному элементу.

## Описание и алгоритм

Программа выполняет следующие шаги:

1. Запрашивает размер матрицы `M x N`;
2. Генерирует матрицу случайными целыми числами в диапазоне от `10` до `99`;
3. Находит максимальный элемент `maxVal = max(matrix)` во всей матрице.
4. Нормализует матрицу, деля каждый элемент `x` на `maxVal`: `x = x / maxVal`.
5. Выводит матрицу:
    * до нормализации (**Before**)
    * после нормализации (**After**) с точностью до двух знаков после запятой.

## Использование

1. Запустите скомпилированный исполняемый файл
2. Введите размер матрицы `M x N`.
3. Программа выведет:
    - Сгенерированную матрицу (Before)
    - Нормализированную матрицу (After)

## Пример работы

Ввод:

```text
Enter number of rows (M) and columns (N): 2 3
```

Вывод (пример):

```text
Before:
12.00 95.00 44.00
80.00 21.00 63.00

After (normalized):
0.13 1.00 0.46
0.84 0.22 0.66
```

Примечание: значения матрицы случайные, поэтому результат при каждом запуске отличается.
