# Метод Симпсона и Трапеций

В этой программе реализованы численные методы интегрирования (приближённого вычисления определённого интеграла).

Интегрируется функция:

$$f(x) = xe^{-x}$$

на отрезке $[a,b]=[−1,π]$

## Алгоритм

### Общая логика программы

Оба метода работают по одной схеме:

1. Берётся начальное разбиение
2. Считается интеграл
3. Уменьшается шаг (увеличивается число разбиений)
4. Сравниваются два соседних результата:
    $$|I_{new} - I_{old}|$$
5. Если ошибка меньше E = 0.00001 → остановка

### 1. Метод Симпсона

Метод Симпсона (парабол) аппроксимирует функцию на каждом интервале $[x_i, x_{i+1}]$ квадратичной параболой. Для этого используются три точки: концы интервала и его середина.

**Формула для одного интервала:**
$$I_i = \frac{h}{6}\left(f(x_i) + 4f\left(\frac{x_i+x_{i+1}}{2}\right) + f(x_{i+1})\right)$$

где $h = \frac{b-a}{n}$ — шаг разбиения.

**Общая формула при чётном числе разбиений $n = 2m$:**
$$I \approx \frac{h}{3}\left[f(x_0) + f(x_{2m}) + 4\sum_{k=1}^{m} f(x_{2k-1}) + 2\sum_{k=1}^{m-1} f(x_{2k})\right]$$

Метод имеет четвёртый порядок точности (погрешность $O(h^4)$).

### 2. Метод Трапеций

Метод трапеций аппроксимирует функцию на каждом интервале прямой линией, соединяющей концы отрезка. Площадь под графиком представляется как сумма трапеций.

**Формула:**
$$I \approx h\left[\frac{f(x_0) + f(x_n)}{2} + \sum_{i=1}^{n-1} f(x_i)\right]$$

Метод имеет второй порядок точности (погрешность $O(h^2)$).

### Сравнение методов

| Метод | Порядок точности | Точность |
|-------|------------------|----------|
| Трапеций | $O(h^2)$ | Ниже |
| Симпсон | $O(h^4)$ | Выше |

Метод Симпсона обычно даёт более точный результат при том же числе разбиений.

## Использование

Программа считает все сама и выводит ответ в консоль
